Название урока: 2. Видео уроки Cisco Packet Tracer. Курс молодого бойца. Простейшая сеть
Старт - конец: 23.06.25 - 23.06.25
Цель урока: повторение ранее изученного материала

Задача: соединить два хоста (компьютера) в простейшую сеть.

Выполнение:
1. Обжать витую пару коннекторами RJ45 (бело-оранжевый, оранжевый, бело-зелёный, синий, бело-синий, зелёный, бело-коричневый, коричневый). (Если сетевая карта старой модели использовать перекрёстный кабель для устройств одного уровня модели OSI).
2. Соединить хосты, концы патч-корда подключить в сетевые порты компьютеров.
3. Настроить статический айпи-адрес. Для двух хостов указать айпи адрес из одной сети:

 PC0: IP - 192.168.1.1; маска - 255.255.255.0
 PC1: IP - 192.168.1.2; маска - 255.255.255.0

 (Центр управления сетями и общим доступом -> Изменение параметров адаптера -> Свойства адаптера -> Протокол IPv4 -> Ввести IP вручную).

Работа в CPT (Cisco Packet Tracer):
1. Два компьютера соединить кабелем перекрёстного сечения (cross-over).
2. Установить им айпи адреса из одной маски сети. PC0 - 192.168.1.1/24, PC1 - 192.168.1.2/24
3. Проверить соединение хостов между собой командой ping.
 Команда для PC0: ping 192.168.1.2 - успешно. 
 Команда для PC1: ping 192.168.1.1 - успешно.
